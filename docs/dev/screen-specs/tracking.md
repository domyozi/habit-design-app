# 習慣トラッキング可視化

習慣ごとの達成率・ストリーク・カレンダービューを提供する読み取り専用の可視化画面。週次・月次統計を一覧し、過去の達成パターンを振り返ることができる。

## 画面項目

| # | 項目名 | 種別 | 必須 | 説明 |
|---|--------|------|------|------|
| 1 | 習慣別達成率進捗バー | progress bar | - | 各習慣の達成率を表示。80%以上=緑、50〜79%=黄、50%未満=赤 <!-- 🔵 --> |
| 2 | ストリーク日数 | display | - | 炎アイコン（🔥）付きで連続達成日数を表示 <!-- 🔵 --> |
| 3 | 週次達成率サマリー | display | - | 今週の習慣達成率を集計表示 <!-- 🔵 --> |
| 4 | 月次達成率サマリー | display | - | 今月の習慣達成率を集計表示 <!-- 🔵 --> |
| 5 | カレンダービュー | calendar | - | 日付ごとの達成状況を色分け表示（全達成=緑・一部=黄・未達成=赤・未記録=グレー） <!-- 🔵 --> |
| 6 | 前月ボタン | button | - | カレンダーを前月に切り替える <!-- 🔵 --> |
| 7 | 次月ボタン | button | - | カレンダーを翌月に切り替える <!-- 🔵 --> |

## バリデーション

なし（読み取り専用画面）

## インタラクション

### 日付タップ → 達成詳細モーダル表示 <!-- 🔵 -->
- カレンダーの日付をタップ → その日の習慣達成詳細（習慣名・達成/未達成）をモーダルで表示

### 月切り替え <!-- 🔵 -->
- 前月ボタン押下 → カレンダーが前月に切り替わり、月次統計も更新
- 次月ボタン押下 → カレンダーが翌月に切り替わり、月次統計も更新

## 状態

| 状態 | 条件 | 表示内容の変化 |
|------|------|---------------|
| ローディング | データ取得中 | スケルトン表示（進捗バー・カレンダー領域） <!-- 🔵 --> |
| 通常 | データ取得完了 | グラフ・カレンダー・統計サマリーを表示 <!-- 🔵 --> |

## 備考

- AC-501〜506 に対応する可視化画面（達成率・カレンダービュー・ストリーク表示）
- ストリーク計算は REQ-502/503 に準拠。当日未達成記録でカウントが0リセット
- 色分け基準（80%以上=緑 / 50〜79%=黄 / 50%未満=赤）はUIの視認性を考慮した推奨値
