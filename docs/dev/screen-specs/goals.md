# 長期目標管理

AI提案により生成された長期目標（Goal）の一覧を管理する画面。目標の追加・編集・削除をCRUD操作で提供する。

## 画面項目

| # | 項目名 | 種別 | 必須 | 説明 |
|---|--------|------|------|------|
| 1 | 目標一覧 | list | - | 登録済み目標の一覧。タイトル・説明・関連習慣数を表示 <!-- 🔵 --> |
| 2 | 目標タイトル | text | - | 各目標のタイトル表示 <!-- 🔵 --> |
| 3 | 目標説明 | text | - | 各目標の説明テキスト表示 <!-- 🔵 --> |
| 4 | 関連習慣数 | badge/text | - | 目標に紐づく習慣の件数表示 <!-- 🔵 --> |
| 5 | 「目標を追加」ボタン | button | - | 新規目標の追加モーダルを開く <!-- 🔵 --> |
| 6 | 目標追加・編集フォーム（モーダル） | modal | - | タイトル・説明の入力フィールドを含む <!-- 🔵 --> |
| 7 | モーダル内タイトル入力 | text input | Yes | 目標タイトルの入力フィールド <!-- 🔵 --> |
| 8 | モーダル内説明入力 | textarea | - | 目標の詳細説明入力フィールド <!-- 🟡 --> |
| 9 | モーダル内「保存」ボタン | button | - | 目標の新規登録または更新を実行 <!-- 🔵 --> |
| 10 | モーダル内「キャンセル」ボタン | button | - | モーダルを閉じる <!-- 🔴 --> |
| 11 | 目標削除ボタン | button | - | 各目標の削除アクションを起動 <!-- 🔵 --> |
| 12 | 削除確認ダイアログ | dialog | - | 削除前に確認を求めるダイアログ <!-- 🔵 --> |
| 13 | 空状態メッセージ | text | - | 目標0件時の案内メッセージ <!-- 🔵 --> |

## バリデーション

| 項目 | ルール | エラーメッセージ |
|------|--------|-----------------|
| 目標タイトル | 必須入力 | タイトルを入力してください <!-- 🔵 --> |
| 目標タイトル | 100文字以内 | タイトルは100文字以内で入力してください <!-- 🔴 --> |

## インタラクション

### 目標追加 <!-- 🔵 -->
- 「目標を追加」ボタン押下 → 追加モーダルが開く → タイトル・説明を入力 → 「保存」押下 → DBに保存 → モーダルを閉じて一覧に追加

### 目標編集 <!-- 🔵 -->
- 目標アイテムをタップ/クリック → 編集モーダルが開き既存値を表示 → 修正 → 「保存」押下 → DBを更新 → モーダルを閉じて一覧を更新

### 目標削除 <!-- 🔵 -->
- 削除ボタン押下 → 確認ダイアログ表示 → 「削除する」確認 → DBから削除 → 一覧から除去

### モーダルキャンセル <!-- 🔴 -->
- 「キャンセル」ボタン押下またはモーダル背景クリック → モーダルを閉じる（変更は破棄）

## 状態

| 状態 | 条件 | 表示内容の変化 |
|------|------|---------------|
| ローディング | データ取得中 | スケルトンローダー表示 <!-- 🔵 --> |
| 通常（目標あり） | 1件以上の目標が存在 | 目標一覧を表示 <!-- 🔵 --> |
| 空状態 | 目標0件 | 「まだ目標がありません。Wanna Be設定からAIに提案してもらいましょう」などの案内メッセージ表示 <!-- 🔵 --> |
| モーダル表示中 | 追加・編集操作中 | モーダルがオーバーレイで表示 <!-- 🟡 --> |
| 保存中 | APIリクエスト中 | 「保存」ボタンdisabled + スピナー <!-- 🔴 --> |
| 削除確認中 | 削除ボタン押下後 | 確認ダイアログ表示 <!-- 🔵 --> |
