# 設定

プロフィール・通知設定の確認と変更、バッジギャラリーの閲覧、およびログアウトを行う設定画面。

## 画面項目

| # | 項目名 | 種別 | 必須 | 説明 |
|---|--------|------|------|------|
| 1 | 表示名（display_name） | text input | No | ユーザーの表示名を編集できる <!-- 🔵 --> |
| 2 | 通知メールアドレス | email input | No | 通知先メールアドレス。メール形式のバリデーションあり <!-- 🔵 --> |
| 3 | 通知オン/オフトグル | toggle | No | メール通知の有効/無効を切り替える <!-- 🔵 --> |
| 4 | 週次レビュー曜日 | select | No | 週次レビューを実施する曜日を選択（月〜日） <!-- 🔵 --> |
| 5 | バッジギャラリー | gallery | - | 獲得済みバッジの一覧表示（BadgeGallery） <!-- 🔵 --> |
| 6 | 保存ボタン | button | - | プロフィール・通知設定を保存する <!-- 🔵 --> |
| 7 | ログアウトボタン | button | - | セッションを終了してログイン画面へ遷移 <!-- 🔵 --> |

## バリデーション

| 項目 | ルール | エラーメッセージ |
|------|--------|-----------------|
| 通知メールアドレス | メール形式 <!-- 🔵 --> | 正しいメールアドレスを入力してください <!-- 🔴 --> |

## インタラクション

### 設定保存 <!-- 🔵 -->
- 保存ボタン押下 → PATCH /users/me へリクエスト → 成功時「保存しました」トーストを表示
- 保存中はボタンをdisabled状態にしスピナーを表示

### ログアウト <!-- 🔵 -->
- ログアウトボタン押下 → supabase.auth.signOut() 実行 → /login へリダイレクト

### バッジ詳細確認 <!-- 🟡 -->
- バッジタップ → バッジ詳細モーダルを表示（名称・獲得条件・獲得日）

## 状態

| 状態 | 条件 | 表示内容の変化 |
|------|------|---------------|
| 初期表示 | ページ読み込み完了 | 現在の設定値（display_name・通知設定・曜日）を各フォームに反映 <!-- 🔵 --> |
| 保存中 | 保存ボタン押下後・APIリクエスト中 | 保存ボタンdisabled + スピナー表示 <!-- 🔵 --> |
| 保存成功 | PATCH 200応答 | 「保存しました」トーストを表示 <!-- 🟡 --> |
| 保存失敗 | PATCH エラー応答 | 「保存に失敗しました」トーストを表示 <!-- 🔵 --> |
