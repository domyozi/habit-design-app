## タスク概要

音声入力UIを実装する。Web Speech APIで音声認識し、テキストに変換してAI分類APIへ送信する。
非対応ブラウザ（Firefox/Safari）ではテキスト入力にフォールバックする（architecture.md 音声入力制約）。
AI判断不能時（EDGE-003）はユーザーに手動選択を促す。

## 注意事項

- Web Speech APIはChrome/Edgeのみ対応。Firefox/Safariは必ずテキスト入力にフォールバックすること
- `webkitSpeechRecognition` と `SpeechRecognition` の両方をチェックすること
- 音声認識の `lang` は `'ja-JP'` に設定すること
