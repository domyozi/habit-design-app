**タスクID**: TASK-0032

**担当**: Codex（`frontend-v2/src/` 修正）
**前提**: バックエンド側の対応は完了済み（コミット前）

## 背景

設定画面のスクリーンショット 3 点に対する改修。バックエンド側はすでに以下を実装済み:

- `user_profiles.age` カラム追加マイグレーション (`backend/migrations/add_age_to_user_profiles.sql`)
- `UserProfile` / `UpdateUserProfileRequest` に `age` フィールドを追加 (`backend/app/models/schemas.py`)
- `POST /api/journals` 時にバックグラウンドで AI メモリ抽出 → `user_context` を追記マージ (`backend/app/api/routes/journal.py`, `backend/app/services/ai_service.py:extract_memory_facts`, `merge_memory_patch`)

フロントエンド側の対応が残っている。
