## タスク概要

Daily OS v3 の起点となる Home 画面を実装する。  
アプリを開いた瞬間に「今何をすべきか」が伝わる画面。

### `pages/HomePage.tsx`

**レイアウト（上から順）**:

1. **時刻・日付バー**
   - 現在時刻（HH:MM）、日付（M/D 曜日）を表示
   - `useTimeContext()` で時刻帯を取得

2. **ContextCard（時刻帯ハイライト）** — `components/home/ContextCard.tsx`
   - 朝帯（5〜12時）: オレンジ背景「🌅 朝ルーティンを始めよう」→ タップで朝タブへ
   - 夜帯（17〜24時）: 紫背景「🌙 夜の振り返りへ」→ タップで夜タブへ
   - その他: グレー「☀️ 今日のまとめ」達成数/全習慣数を表示

3. **BossCard** — `components/home/BossCard.tsx`（v2 のボスカードを移植）
   - `useBossStorage()` で今日のボスを取得
   - タップで達成トグル
   - 未設定時: 「夜タブで明日のボスを設定 →」

4. **ProgressRings セクション**
   - MUST習慣4件（早起き/筋トレ/英語/有酸素）のリングを2×2グリッドで表示
   - `countMonthlyChecks('morning:checked')` からデータ取得
   - 各リングに ProgressRing コンポーネントを使用

5. **今日の達成サマリー**
   - 「今日 N / M 完了」テキスト
   - `useTodayStorage('morning:checked', [])` から算出
