## タスク概要

Home画面の中核となる進捗リングを実装する。  
棒グラフではなくSVGリングで「目標 / 実績 / ベスト」の3軸を表現し、モバイルで一瞬で把握できる表示にする。

### `components/home/ProgressRing.tsx`

**Props**:
```ts
interface ProgressRingProps {
  habitId: string       // 集計対象の習慣ID
  label: string         // 表示ラベル（例: "筋トレ"）
  color: string         // リング色（例: "#ff6b35"）
  target: number        // 今月の目標回数
  actual: number        // 今月の実績回数（countMonthlyChecks から取得）
  best?: number         // 過去最高（optional）
  size?: number         // リング直径（デフォルト 80px）
}
```

**表示仕様**:
- 外円: ベスト値（灰色、薄い）
- 中円: 目標値（白/10）
- 内塗り: 実績値（color）
- 中央テキスト: 実績数 / 目標数
- 下部テキスト: ラベル名
- 実績が目標超過の場合: リングを閉じて「✅」表示

**SVG 仕様**:
- `stroke-dasharray` / `stroke-dashoffset` でパーセントを描画
- アニメーション: `transition-all duration-500`
