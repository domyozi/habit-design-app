## 実装詳細

### 1. ダッシュボードページ 🔵

**信頼性**: 🔵 *REQ-306 より*

**実装ファイル**: `frontend/src/pages/Dashboard.tsx`

- TanStack Queryで `GET /habits?include_today_log=true` を取得
- 週間統計を表示（`WeeklyStats` コンポーネント）
- 習慣リストを表示（`HabitList` コンポーネント）
- グローバルナビゲーション（ダッシュボード / Wanna Be / 週次レビュー / 設定）

---

### 2. 習慣リストコンポーネント 🔵

**信頼性**: 🔵 *REQ-205/306/502 より*

**実装ファイル**: `frontend/src/components/habits/HabitList.tsx`

- 習慣ごとに以下を表示:
  - 習慣タイトル
  - Wanna Be接続文（`wanna_be_connection_text`、例: 「→ 健康的な体づくりに +1」、REQ-205）
  - ストリーク日数（🔥N日連続、REQ-502）
  - チェックボックス（タップ領域44×44px、NFR-202）
  - 予定時刻（`scheduled_time` が設定されている場合）

---

### 3. 週間統計コンポーネント 🔵

**信頼性**: 🔵 *REQ-504/505 より*

**実装ファイル**: `frontend/src/components/dashboard/WeeklyStats.tsx`

- 今週の達成率（%）テキスト表示
- 習慣ごとの進捗バー（Tailwind CSS `w-{n}/12` で実装）
- 達成数 / 総習慣数の表示

---

### 4. スケルトンローダー 🟡

**信頼性**: 🟡 *UX向上のための推測による実装*

**実装ファイル**: `frontend/src/components/ui/Skeleton.tsx`

- データ取得中はスケルトンアニメーション（`animate-pulse`）を表示
