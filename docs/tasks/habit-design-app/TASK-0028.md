## タスク概要

Daily OS v3 の起点となる基盤コンポーネントを構築する。

1. **`lib/timeContext.ts`** — 時刻帯判定ロジック
   - 朝（5:00〜11:59）/ 夜（17:00〜23:59）/ その他（昼・深夜）を判定
   - `useTimeContext()` フックを export
   - `TimeContext` 型: `{ period: 'morning' | 'evening' | 'other', hour: number, label: string }`

2. **`components/layout/BottomNav.tsx`** — 4項目ボトムナビ
   - Home / 朝ルーティン / 夜の振り返り / その他 の4タブ
   - 現在の TabBar（5タブ）を置き換える
   - アクティブタブはアンバー下線で示す
   - アイコン: 🏠 / 🌅 / 🌙 / ☰

3. **`App.tsx` 構造変更**
   - TabBar → BottomNav に差し替え
   - デフォルトタブを `'home'` に変更（`TabId` 型に追加）
   - `HomePage` コンポーネントのプレースホルダーを追加（中身は TASK-0030 で実装）
