### 4. FastAPI アプリケーション最小構成 🔵

**信頼性**: 🔵 *architecture.md バックエンドコンポーネント構成より*

`backend/app/main.py` を以下の内容で作成する。

```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from dotenv import load_dotenv
import os

load_dotenv()

app = FastAPI(
    title="Habit Design App API",
    description="未来の自分から逆算して習慣を設計・トラッキングするアプリのAPI",
    version="0.1.0",
)

# CORS設定
frontend_url = os.getenv("FRONTEND_URL", "http://localhost:5173")
app.add_middleware(
    CORSMiddleware,
    allow_origins=[frontend_url],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)


@app.get("/")
async def root():
    return {"message": "Habit Design App API is running"}


@app.get("/health")
async def health_check():
    return {"status": "ok"}
```

**実装ファイル**: `backend/app/main.py`

## 注意事項

- Python 3.11 以上を使用すること（`python3 --version` で確認）
- 仮想環境 `.venv/` はバージョン管理対象外とすること（`.gitignore` に追加）
- 実際の API キーを含む `.env` はコミットしないこと。`.env.example` のみコミット対象
- `uvicorn` の起動は必ず `backend/` ディレクトリ内から実行すること
- Pydantic v2 を使用するため、v1 系の記法（`class Config:` など）は使用しないこと
