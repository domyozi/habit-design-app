## 実装詳細

### 1. Resendメール送信サービス 🔵

**信頼性**: 🔵 *REQ-801・design-interview.md Q4（Resend選定）より*

**実装ファイル**: `backend/app/services/email_service.py`

```python
import resend
from app.core.config import settings

resend.api_key = settings.RESEND_API_KEY

async def send_reminder_email(to_email: str, user_name: str) -> bool:
    """週次レビューリマインダーメール送信"""
    try:
        resend.Emails.send({
            "from": "habit-coach@{your-domain}",
            "to": to_email,
            "subject": "今週の振り返りをしましょう",
            "html": f"<p>今週の習慣を振り返りましょう。</p>"
        })
        return True
    except Exception:
        return False  # メール失敗はログのみ、アプリは継続
```

### 2. 週次レビューリマインダースケジューラー 🔵

**信頼性**: 🔵 *REQ-701・architecture.md APScheduler より*

**実装ファイル**: `backend/scheduler/weekly_review.py`

```python
from apscheduler.schedulers.asyncio import AsyncIOScheduler
from apscheduler.triggers.cron import CronTrigger

scheduler = AsyncIOScheduler()

async def send_weekly_reminders():
    """毎日実行: 本日が週次レビュー曜日のユーザーにリマインダー送信"""
    from datetime import date
    today_weekday = date.today().isoweekday()  # 1=月〜7=日
    # user_profilesから本日がweekly_review_dayのユーザーを取得
    # notification_enabled=trueのユーザーのみ（REQ-802）
    # send_reminder_emailでメール送信

scheduler.add_job(
    send_weekly_reminders,
    CronTrigger(hour=8, minute=0),  # 毎日8時に実行
    id="weekly_review_reminder"
)
```

### 3. FastAPI lifespan統合 🔵

**信頼性**: 🔵 *architecture.md APScheduler統合より*

**実装ファイル**: `backend/app/main.py`（lifespan更新）

```python
from contextlib import asynccontextmanager
from scheduler.weekly_review import scheduler

@asynccontextmanager
async def lifespan(app: FastAPI):
    scheduler.start()
    yield
    scheduler.shutdown()

app = FastAPI(lifespan=lifespan)
```
