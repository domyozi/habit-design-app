## 実装詳細

### 1. 設定ページ 🔵

**信頼性**: 🔵 *REQ-701/801/802 より*

**実装ファイル**: `frontend/src/pages/Settings.tsx`

React Hook Formでフォーム管理。以下のセクションを含む:

**プロフィールセクション**:
- ユーザー名（display_name）表示・編集

**通知設定セクション**:
- 通知メールアドレス（input type="email"）（REQ-801）
- 通知オン/オフトグルスイッチ（REQ-802）
- 週次レビュー曜日（select: 月〜日）（REQ-701）

**バッジセクション**:
- `BadgeGallery` コンポーネント組み込み（TASK-0022）

**ログアウトボタン**:
- `supabase.auth.signOut()` を呼び出し → `/login` へリダイレクト

**保存ボタン**:
- `PATCH /users/me` で設定を保存
